# Brief
Simple  data transfer between SPI1, SPI2, both in one STM32 board.
Watch SPI_data_send, SPI_data_get using STMStudio.
# Pinouts 
SPI1 <-> SPI2
PA4 <-> PB12  
PA5 <-> PB13
PA6 <-> PB14
PA7 <-> PB15
